## DSP — the sound

### Character / coloration algorithms

The "vibe" algorithms — harvested from shipping devices, each proven in Live. Self-contained: emit their own state, namespaced by `prefix`, hoist decls to the top.

`m4l_builder.gen_character` — 11 functions

- **`optical_leveler`** — LA-2A-style optical leveler with a *program-dependent* release (harvested from Opti)
- **`wow_flutter`** — Tape transport drift as a modulated read position in SAMPLES (from Chrono)
- **`tape_hysteresis`** — Magnetic-tape record-head saturation with hysteresis memory (from Chrono)
- **`leslie_rotor`** — Leslie rotary speaker: dual horn+drum rotors with mechanical inertia (from Rotary)
- **`bbd_chorus`** — Bucket-brigade string-machine chorus, 2 or 3 voices (from Ensemble)
- **`deesser`** — Split-band de-esser: duck only the high band when sibilance triggers (from Sibilant Surgeon)
- **`console_saturation`** — Asymmetric console/transformer soft-clip with DC blocking (from Console)
- **`crosstalk`** — Analog channel crosstalk: each channel bleeds a low-weighted copy of the OTHER into itself (from Console)
- **`iron_transformer`** — Transformer "iron" tone: a low-frequency bump + a gentle top dip, one knob (from Console)
- **`duck_gain`** — Sidechain ducking gain: 1.0 when the trigger is quiet, pulled down while it is hot (from Ducking Delay)
- **`haas_widener`** — Haas-effect stereo widener: delay ONE channel a few ms for width (from Expansion)

### Neutral GenExpr primitives

Clean building blocks — filters, dynamics gain-computers, saturators, M/S, loudness, oversampling, fractional delay reads.

- **`ms_encode`** — Encode L/R -> Mid/Side: ``mid = (L+R)/2``, ``side = (L-R)/2``
- **`ms_decode`** — Decode Mid/Side -> L/R: ``L = M+S``, ``R = M-S`` (inverse of ms_encode)
- **`ms_width`** — Stereo-width via M/S: scale the Side by ``width`` then decode

- **`tanh_adaa`** — First-order antiderivative-anti-aliased (ADAA) tanh saturation
- **`hardclip_adaa`** — First-order antiderivative-anti-aliased HARD clipper (clamp at +/-1)
- **`square_adaa`** — First-order antiderivative-anti-aliased SQUARER (``x^2``)
- **`peak_follower`** — Attack/release peak envelope follower (the dynamics detector core)
- **`isp_catmull_4x`** — 4x inter-sample-peak (ISP) estimate for one channel via cubic Catmull-Rom
- **`kweight_coeffs_bs1770`** — ITU-R BS.1770-4 K-weighting biquad coefficients, computed at the live rate
